## Product Portfolio

### Current Products

#### 1. Surplusx Platform
**Status:** In Development (MVP Phase)

**Description:**
Comprehensive food waste management platform connecting donors, NGOs, and recipients to reduce food waste and provide nutrition to communities.

**Key Features:**
- Donor dashboard for food surplus listing
- NGO coordination and logistics
- Recipient (beneficiary) management
- Real-time tracking and notifications
- AI-driven freshness scoring
- Impact metrics and analytics

**Technology Stack:**
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Python + FastAPI
- **Database:** PostgreSQL
- **Authentication:** JWT-based auth system
- **Deployment:** Docker + Kubernetes (planned)
- **Infrastructure:** Cloud-ready architecture

**Team Allocation:**
- Backend: Matang Mehra (50%), Saksham Rastogi (20%)
- Frontend: Ramit Arora (50%), Design support (20%)
- Design: Priya Singh (80%)

**Launch Timeline:** Q2-Q3 2026
